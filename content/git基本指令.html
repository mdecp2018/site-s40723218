git基本指令

git clone  將遠端檔案下載至近端

git status 查詢有更動的資料

git add .  增加

git commit -m "initial add"  提交近端檔案並命名為""內的字

git config --global user_email  只有第一次上傳檔案需要登入，儲存至home

git config --global user_name  s學號  同上

git push 將近端檔案上傳遠端

dir 顯示目錄的資料夾

python wsgi.py 打開近端網站必須

git checkout gh-pages  導出gh-page

git pull 近端遠端同步

mkdir 名稱 建立目錄
